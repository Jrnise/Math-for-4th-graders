P.E.M.D.A.S Explanation

P.E.M.D.A.S is is order of operations. P.E.M.D.A.S is an easy way to remember the way to solve equations in the correct order. This is what it stands for: P: Parentheses E: Exponents M: Multiplication D: Division (from left to right) A: Addition S: Subtraction (from left to right)

Place value explanation

Place value is the value of each digit in a number. For example, the 5 in 350 represents 5 tens, or 50; however, the 5 in 5,006 represents 5 thousands, or 5,000
